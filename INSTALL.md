# Installation Guide

## Quick Setup

```bash
git clone https://github.com/TARVIA-lab/tarvia-validation.git
cd tarvia-validation
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python verify_setup.py
```

## Requirements

- Python 3.10+

## Installation

```bash
pip install -r requirements.txt
```

## Verify

```bash
python verify_setup.py
```
